Glassmorphism Galaxy Navbar

A modern, responsive navigation bar built with HTML and CSS, featuring a glassmorphism effect and a galaxy-inspired design. The navbar uses blurred transparent backgrounds, glowing accents, and smooth hover animations to create a futuristic user interface.

Features
✨ Glassmorphism (blurred translucent background)
🌌 Galaxy-themed visual effects
📱 Responsive design
🎨 Smooth hover and transition animations
⚡ Lightweight and pure CSS
🔧 Easy customization
Technologies Used
HTML5
CSS3 (Backdrop Filter, Gradients, Animations)
Usage
Clone or download the project.
Open index.html in your browser.
Customize colors, blur intensity, and animations in style.css.
Customization
Change galaxy colors using CSS gradients.
Adjust glass effect with backdrop-filter.
Modify hover effects and transition speeds.
Preview

A sleek navigation bar with a frosted-glass appearance, glowing galaxy accents, and smooth interactive effects.
