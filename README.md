# Glassmorphism Navbar

A centered glassmorphism navbar with smooth sliding pill animation and a galaxy-inspired CSS gradient background. No libraries, no frameworks — pure HTML, CSS & JavaScript.

## ✨ Features
- Centered layout — vertical & horizontal both
- Frosted glass effect with backdrop blur
- Smooth sliding pill animation on nav links
- Galaxy-inspired CSS gradient background (no image needed)
- Zero dependencies

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript (vanilla)

## 🚀 How it works
The sliding pill is a `div` that moves using JavaScript `offsetLeft` and `offsetWidth` to match the active link position. Smooth animation is handled via CSS `transition` and `cubic-bezier`.

The background uses layered `radial-gradient` blobs with `filter: blur()` to create a glowing galaxy effect — no images required.

## 📂 Usage
Just open `navbar3.html` directly in your browser. No setup needed.
